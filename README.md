# Dify Chatbot DSL

このリポジトリは、Difyで作成したチャットボットのDSL(Domain Specific Language)定義ファイルを管理しています。

## 概要

Difyのチャットボット設定をYAML形式のDSLファイルとして保存し、バージョン管理することで以下のメリットが得られます:

- 変更履歴の追跡
- ロールバックの容易化

## ディレクトリ構造

```
.
├── README.md
├── chatbot/
│   ├── DSL.yml 
│   ├── README.md
│   └── ...
└── docs/
    └── ...
```

## ブランチ戦略

- `main`: 安定版
- `feature/*`: 新機能開発用ブランチ

## コミットメッセージ規約

```
Add: 新規追加
Update: 既存機能の更新
Fix: バグ修正
Docs: ドキュメントの更新
```

## タグ

週報を提出するタイミングで、`YYYYMMDD週報提出`という名称の`報告用`タグを作成してください


## 注意事項

### 機密情報の取り扱い

- **APIキー、トークン、パスワードなどの機密情報をDSLファイルに含めないでください**
- 環境変数や外部設定ファイルで管理してください
- `.gitignore`に機密情報を含むファイルを追加してください
